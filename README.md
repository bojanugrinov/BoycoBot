## ✨ Features

- **Modular Command System** - Easily add, manage, and organize commands
- **TypeScript** - Full type safety and modern JavaScript features
- **Slash Commands** - Modern Discord slash command support
- **Interaction Handling** - Robust event and interaction handling
- **Environment Configuration** - Secure configuration with environment variables

## 🚀 Quick Start

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- A Discord application with a bot token

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/bojanugrinov/BoycoBot.git
   cd BoycoBot
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Create a `.env` file** in the root directory

   ```env
   DISCORD_TOKEN=your_bot_token_here
   CLIENT_ID=your_client_id_here
   GUILD_ID=your_guild_id_here
   ```

4. **Run the bot**
   ```bash
   npm run dev
   ```

## 📦 Available Scripts

- `npm run build` - Compile TypeScript to JavaScript
- `npm start` - Build and run the bot
- `npm run dev` - Run the bot in development mode with hot reload

## 📚 Technology Stack

- **[Discord.js](https://discord.js.org/)** - JavaScript Discord API wrapper
- **[TypeScript](https://www.typescriptlang.org/)** - JavaScript superset for type safety
- **[Node.js](https://nodejs.org/)** - JavaScript runtime
